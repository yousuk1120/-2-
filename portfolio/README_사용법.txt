==========================================
포트폴리오 웹 페이지 열기 방법
==========================================

【 방법 1: 가장 쉬운 방법 】
1. 이 폴더에서 "열기.bat" 파일을 더블클릭하세요
   → 자동으로 브라우저에서 열립니다!

【 방법 2: 직접 파일 열기 】
1. "index.html" 파일을 찾으세요
2. 파일을 우클릭 → "연결 프로그램" → 웹 브라우저 선택
   또는 파일을 브라우저 창으로 드래그하세요

【 방법 3: 서버로 실행하기 】
PowerShell 또는 명령 프롬프트에서:

   cd portfolio
   python -m http.server 8000

그 다음 브라우저에서 http://localhost:8000 접속

==========================================
주의사항
==========================================
- 방법 1과 2는 서버 없이 바로 작동합니다
- 모든 파일(index.html, styles.css, script.js)이 같은 폴더에 있어야 합니다
- 브라우저는 Chrome, Edge, Firefox 등 최신 브라우저를 사용하세요
