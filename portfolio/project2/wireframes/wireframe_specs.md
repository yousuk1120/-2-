# 프로젝트 2: 와이어프레임 상세 명세

## 와이어프레임 1: 장바구니 담기 직후 모달

### 목적
즉시 구매 유도 및 할인 쿠폰 제공

### 레이아웃 구조
```
┌─────────────────────────┐
│  [반투명 배경]          │
│                         │
│  ┌─────────────────┐   │
│  │                 │   │
│  │  ✓ 장바구니에   │   │
│  │    담겼습니다   │   │
│  │                 │   │
│  │  🎁 10% 할인    │   │
│  │     쿠폰 지급!  │   │
│  │                 │   │
│  │  [계속 쇼핑하기] │   │
│  │  [장바구니 보기] │   │
│  │                 │   │
│  │  [지금 구매하기] │   │
│  │  (강조 버튼)    │   │
│  │                 │   │
│  └─────────────────┘   │
│                         │
└─────────────────────────┘
```

### 구성 요소
- 모달 배경 (반투명)
- 모달 내용:
  - "장바구니에 담겼습니다" 메시지
  - 할인 쿠폰 제공 안내
  - [장바구니 보기] / [계속 쇼핑하기] 버튼
  - [지금 구매하기] 버튼 (강조)

- [구매하기] 버튼

## 와이어프레임 3: 장바구니 페이지 (개선 후)

### 목적
명확한 정보 제공 및 구매 유도

### 레이아웃 구조
```
┌─────────────────────────┐
│ 장바구니    [전체 삭제] │
├─────────────────────────┤
│                         │
│ ┌──────────────────┐   │
│ │ [큰 이미지]      │   │
│ │ 상품명 전체 표시  │   │
│ │ 옵션: 명확히 표시│   │
│ │ ~~20,000원~~     │   │
│ │ 15,000원 (25%↓)  │   │
│ │ [수량 ▼] [삭제]  │   │
│ │ [위시리스트]     │   │
│ └──────────────────┘   │
│                         │
│ ┌──────────────────┐   │
│ │ [큰 이미지]      │   │
│ │ 상품명 전체 표시  │   │
│ │ 옵션: 명확히 표시│   │
│ │ 25,000원         │   │
│ │ [수량 ▼] [삭제]  │   │
│ │ [위시리스트]     │   │
│ └──────────────────┘   │
│                         │
│ ┌──────────────────┐   │
│ │ 🎁 적용 가능한   │   │
│ │    쿠폰 3개      │   │
│ │ [쿠폰 적용하기]  │   │
│ └──────────────────┘   │
│                         │
├─────────────────────────┤
│ 상품 금액: 40,000원     │
│ 배송비: 무료            │
│ (무료 배송까지 10,000원)│
│ 할인 금액: -5,000원     │
│ ───────────────────     │
│ 최종 결제 금액          │
│ 35,000원 (강조)         │
│                         │
│    [구매하기]           │
│    (큰 버튼, 강조)      │
└─────────────────────────┘
```

### 구성 요소
- 상단: 페이지 타이틀, [전체 삭제] 버튼
- 상품 리스트 (카드 형태, 개선)
  - 상품 이미지 (크기 확대)
  - 상품명 (전체 표시)
  - 옵션 정보 (명확히 표시)
  - 가격 정보 (할인 전/후 비교)
  - 수량 변경 (큰 버튼)
  - [삭제] / [위시리스트] 버튼
- 중간: 할인/쿠폰 정보 배너
  - 적용 가능한 쿠폰 개수 표시
  - [쿠폰 적용하기] 버튼
- 하단 고정: 금액 요약
  - 상품 금액
  - 배송비 (무료 배송까지 금액 표시)
  - 할인 금액
  - 최종 결제 금액 (강조)
- [구매하기] 버튼 (큰 버튼, 강조)

## 와이어프레임 4: 결제 1단계 - 주소 및 배송

### 목적
배송 정보 입력 간소화

### 레이아웃 구조
```
┌─────────────────────────┐
│ 결제 (1/3)              │
├─────────────────────────┤
│                         │
│ 배송지 선택              │
│                         │
│ ┌──────────────────┐   │
│ │ 🏠 집             │   │
│ │ 서울시 강남구...  │   │
│ │ [선택] [수정]    │   │
│ └──────────────────┘   │
│                         │
│ ┌──────────────────┐   │
│ │ 🏢 회사           │   │
│ │ 서울시 서초구...  │   │
│ │ [선택] [수정]    │   │
│ └──────────────────┘   │
│                         │
│ [+ 새 주소 추가]        │
│                         │
├─────────────────────────┤
│ 배송 옵션                │
│ ○ 일반 배송 (무료)      │
│   2-3일 소요            │
│ ○ 빠른 배송 (+3,000원)  │
│   다음날 도착           │
│                         │
│ 배송 요청사항            │
│ [문 앞에 놓아주세요]     │
│                         │
│      [다음 단계]         │
└─────────────────────────┘
```

### 구성 요소
- 상단: 진행 단계 표시 (1/3)
- 주소록 섹션
  - 저장된 주소 리스트 (카드 형태)
  - [새 주소 추가] 버튼
- 배송 옵션 선택
  - 일반 배송 / 빠른 배송
  - 배송비 정보
- 배송 요청사항 입력
- [다음 단계] 버튼

## 와이어프레임 5: 결제 2단계 - 결제 정보

### 목적
결제 수단 및 할인 정보 선택

### 레이아웃 구조
```
┌─────────────────────────┐
│ 결제 (2/3)              │
├─────────────────────────┤
│                         │
│ 결제 수단                │
│ ○ 신용카드              │
│   [카드 선택 ▼]         │
│ ○ 계좌이체              │
│ ○ 간편결제              │
│   [카카오페이] [네이버페이]│
│                         │
├─────────────────────────┤
│ 할인 및 적립             │
│                         │
│ 쿠폰                     │
│ [쿠폰 선택 ▼]           │
│ 적용 가능: 3개           │
│ 할인: -5,000원          │
│                         │
│ 포인트                   │
│ 보유: 5,000P            │
│ 사용: [0] P             │
│                         │
├─────────────────────────┤
│ 결제 금액 요약           │
│ 상품: 40,000원          │
│ 배송비: 무료            │
│ 할인: -5,000원          │
│ ───────────────────     │
│ 최종 결제: 35,000원     │
│                         │
│      [다음 단계]         │
└─────────────────────────┘
```

### 구성 요소
- 상단: 진행 단계 표시 (2/3)
- 결제 수단 선택
  - 카드, 계좌이체, 간편결제 등
  - 자주 사용한 결제 수단 상단 표시
- 쿠폰 선택
  - 적용 가능한 쿠폰 리스트
  - 쿠폰별 할인 금액 표시
- 포인트 사용
  - 보유 포인트 표시
  - 사용할 포인트 입력
- 최종 결제 금액 요약
- [다음 단계] 버튼

## 와이어프레임 7: 결제 완료 페이지

### 목적
구매 완료 확인 및 다음 액션 유도

### 레이아웃 구조
```
┌─────────────────────────┐
│                         │
│        ✓               │
│                         │
│  주문이 완료되었습니다   │
│                         │
│  주문번호: 20240101001  │
│                         │
│  배송 예정일            │
│  2024년 1월 3일 (수)    │
│                         │
│  [주문 상세 보기]       │
│                         │
├─────────────────────────┤
│ 이런 상품은 어때요?      │
│                         │
│ ┌──┐ ┌──┐ ┌──┐         │
│ │사│ │사│ │사│         │
│ │진│ │진│ │진│         │
│ └──┘ └──┘ └──┘         │
│                         │
├─────────────────────────┤
│ [홈으로] [주문 내역 보기]│
└─────────────────────────┘
```

### 구성 요소
- 체크 아이콘 및 "주문이 완료되었습니다" 메시지
- 주문 번호
- 배송 예정일 정보
- 주문 상세 보기 링크
- [홈으로] / [주문 내역 보기] 버튼
- 관련 상품 추천 섹션
