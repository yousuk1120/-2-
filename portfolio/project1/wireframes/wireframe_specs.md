# 프로젝트 1: 와이어프레임 상세 명세

## 와이어프레임 1: 온보딩 화면

### 목적
사용자 취향 파악 및 첫인상 형성

### 레이아웃 구조
```
┌─────────────────────────┐
│                         │
│      [앱 로고]          │
│                         │
│   TasteMap              │
│   당신만의 맛집을        │
│   발견하세요             │
│                         │
│  취향을 선택해주세요     │
│                         │
│  ┌────┐ ┌────┐ ┌────┐  │
│  │한식│ │양식│ │일식│  │
│  └────┘ └────┘ └────┘  │
│  ┌────┐ ┌────┐ ┌────┐  │
│  │중식│ │카페│ │분위기│ │
│  └────┘ └────┘ └────┘  │
│  ┌────┐ ┌────┐ ┌────┐  │
│  │가성비│ │데이트│ │가족│ │
│  └────┘ └────┘ └────┘  │
│                         │
│      [다음] 버튼        │
│                         │
└─────────────────────────┘
```

### 구성 요소
- 상단: 앱 로고 및 브랜드명
- 중간: 서비스 소개 문구
- 하단: 취향 태그 선택 버튼 (최소 3개 선택)
- 최하단: 다음 버튼

## 와이어프레임 2: 홈 화면

### 목적
개인화된 맛집 추천 제공

### 레이아웃 구조
```
┌─────────────────────────┐
│ [위치] 서울 강남구  [검색]│
├─────────────────────────┤
│                         │
│ 큐레이터 추천            │
│ ┌──┐ ┌──┐ ┌──┐ ┌──┐    │
│ │사│ │사│ │사│ │사│ →  │
│ │진│ │진│ │진│ │진│    │
│ └──┘ └──┘ └──┘ └──┘    │
│                         │
│ 업무 후 추천             │
│ ┌──────────┐            │
│ │ 맛집 카드 │            │
│ │ 이름/거리 │            │
│ │ 큐레이터  │            │
│ └──────────┘            │
│                         │
│ 내 취향 기반 추천        │
│ ┌──┐ ┌──┐ ┌──┐         │
│ │사│ │사│ │사│         │
│ │진│ │진│ │진│         │
│ └──┘ └──┘ └──┘         │
│                         │
├─────────────────────────┤
│ 홈 탐색 저장 프로필      │
└─────────────────────────┘
```

### 구성 요소
- 상단: 위치 정보 및 검색 바
- 큐레이터 추천 섹션 (가로 스크롤)
- 상황별 추천 (업무 후, 데이트, 가족 모임)
- 내 취향 기반 추천
- 하단 네비게이션 바

## 와이어프레임 3: 맛집 상세 페이지

### 목적
맛집에 대한 상세 정보 제공

### 레이아웃 구조
```
┌─────────────────────────┐
│ [←]        [저장] [공유]│
├─────────────────────────┤
│                         │
│   [맛집 사진 캐러셀]     │
│                         │
├─────────────────────────┤
│ 맛집 이름 ⭐ 4.5        │
│ 위치: 서울 강남구        │
│ 영업시간: 11:00-22:00   │
├─────────────────────────┤
│ 큐레이터 리뷰            │
│ ┌──────────────────┐   │
│ │ 큐레이터 프로필   │   │
│ │ "분위기 좋고 맛도 │   │
│ │  좋아요!"         │   │
│ └──────────────────┘   │
├─────────────────────────┤
│ 메뉴 및 가격            │
│ - 메뉴1: 15,000원      │
│ - 메뉴2: 18,000원      │
├─────────────────────────┤
│ 실시간 정보             │
│ 웨이팅: 없음            │
│ 예약: 가능              │
│                         │
├─────────────────────────┤
│ [경로 보기] [저장]      │
│      [공유하기]         │
└─────────────────────────┘
```

### 구성 요소
- 상단: 뒤로가기, 저장, 공유 버튼
- 맛집 사진 캐러셀
- 기본 정보 (이름, 평점, 위치, 영업시간)
- 큐레이터 리뷰 섹션
- 메뉴 및 가격 정보
- 실시간 웨이팅 정보
- 하단 고정: 액션 버튼

│ 추천 맛집 (12개)        │

│ ┌──┐ ┌──┐ ┌──┐         │

│ │진│ │진│ │진│         │
│ └──┘ └──┘ └──┘         │
│ ┌──┐ ┌──┐ ┌──┐         │
│ │사│ │사│ │사│         │
│ │진│ │진│ │진│         │
│ └──┘ └──┘ └──┘         │
│                         │

### 구성 요소

## 와이어프레임 5: 검색 화면

### 목적
사용자가 원하는 맛집을 직접 검색

### 레이아웃 구조
```
┌─────────────────────────┐
│ [←]  [검색 바]          │
├─────────────────────────┤
│                         │
│ 인기 검색어              │
│ #강남 맛집               │
│ #데이트 코스             │
│ #가성비 맛집             │
│                         │
├─────────────────────────┤
│ 필터                     │
│ 지역: [서울 ▼]          │
│ 가격대: [전체 ▼]        │
│ 분위기: [전체 ▼]        │
│ 메뉴: [전체 ▼]          │
│                         │
├─────────────────────────┤
│ 검색 결과 (23개)        │
│                         │
│ ┌──────────────────┐   │
│ │ [사진] 맛집 이름 │   │
│ │ 위치/거리        │   │
│ │ 큐레이터 리뷰    │   │
│ └──────────────────┘   │
│                         │
└─────────────────────────┘
```

### 구성 요소
- 상단: 검색 바
- 인기 검색어
- 필터 옵션
- 검색 결과 리스트

## 와이어프레임 6: 저장 화면

### 목적
사용자가 저장한 맛집 관리

### 레이아웃 구조
```
┌─────────────────────────┐
│ 저장                    │
├─────────────────────────┤
│ [위시리스트] [방문기록] │
│ [내 리뷰]               │
├─────────────────────────┤
│                         │
│ 위시리스트 (8개)        │
│                         │
│ ┌──────────────────┐   │
│ │ [사진] 맛집 이름 │   │
│ │ 위치/거리        │   │
│ │ [삭제] [공유]    │   │
│ └──────────────────┘   │
│                         │
│ ┌──────────────────┐   │
│ │ [사진] 맛집 이름 │   │
│ │ 위치/거리        │   │
│ │ [삭제] [공유]    │   │
│ └──────────────────┘   │
│                         │
├─────────────────────────┤
│ 홈 탐색 저장 프로필      │
└─────────────────────────┘
```

### 구성 요소
- 상단: 페이지 타이틀
- 탭: 위시리스트 / 방문 기록 / 내 리뷰
- 맛집 리스트 (카드 형태)
- 필터 및 정렬 옵션
- 하단 네비게이션 바

## 와이어프레임 7: 프로필 화면

### 목적
사용자 설정 및 개인 정보 관리

### 레이아웃 구조
```
┌─────────────────────────┐
│ 프로필                  │
├─────────────────────────┤
│                         │
│   [프로필 사진]          │
│      사용자 이름         │
│                         │
├─────────────────────────┤
│ 내 취향                  │
│ #한식 #양식 #데이트      │
│ [수정]                   │
├─────────────────────────┤
│ 팔로우한 큐레이터 (5명)  │
│ ┌──┐ ┌──┐ ┌──┐         │
│ │프│ │프│ │프│         │
│ │로│ │로│ │로│         │
│ │필│ │필│ │필│         │
│ └──┘ └──┘ └──┘         │
├─────────────────────────┤
│ 설정                     │
│ 알림 설정                │
│ 개인정보 설정            │
│ 고객센터                 │
│ 로그아웃                 │
│                         │
├─────────────────────────┤
│ 홈 탐색 저장 프로필      │
└─────────────────────────┘
```

### 구성 요소
- 프로필 사진 및 이름
- 내 취향 태그
- 팔로우한 큐레이터
- 설정 메뉴
- 로그아웃
- 하단 네비게이션 바
