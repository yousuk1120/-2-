# 프로젝트 3: 와이어프레임 상세 명세

## 와이어프레임 1: 알림 메시지 (개선 전)

### 목적
개선 전 상태 기록

### 레이아웃 구조
```
┌─────────────────────────┐
│ [앱 아이콘]              │
│                         │
│ 특가 상품을 확인하세요!  │
│                         │
│ 지금 바로 확인해보세요   │
│                         │
│ 발송 시간: 오전 10시     │
│ (모든 사용자 동일)       │
└─────────────────────────┘
```

### 구성 요소
- 앱 아이콘
- 제목: "특가 상품을 확인하세요!" (일반적)
- 본문: "지금 바로 확인해보세요" (과장된 표현)
- 발송 시간: 모든 사용자에게 동일 시간

## 와이어프레임 2: 알림 메시지 (개선 후 - 개인화)

### 목적
개인화된 알림으로 오픈율 향상

### 레이아웃 구조
```
┌─────────────────────────┐
│ [앱 아이콘]              │
│                         │
│ [사용자명]님,            │
│ 관심 상품이 특가예요!     │
│                         │
│ 최근 본 [카테고리]       │
│ 상품 30% 할인            │
│                         │
│ [상품 이미지]            │
│                         │
│ 발송 시간: 사용자별      │
│ 최적 시간 (저녁 7시)     │
└─────────────────────────┘
```

### 구성 요소
- 앱 아이콘
- 제목: "[사용자명]님, 관심 상품이 특가예요!" (개인화)
- 본문: "최근 본 [카테고리] 상품 30% 할인" (구체적 정보)
- 이미지: 상품 이미지 (있는 경우)
- 발송 시간: 사용자별 최적 시간

---

## 와이어프레임 3: 알림 메시지 (개선 후 - 상황 기반)

### 목적
상황에 맞는 알림 제공

### 레이아웃 구조
```
┌─────────────────────────┐
│ [앱 아이콘]              │
│                         │
│ ⚠️ 장바구니 상품이       │
│    곧 품절돼요!          │
│                         │
│ [상품명] 재고 소진 임박  │
│ 지금 구매하세요          │
│                         │
│ [상품 이미지]            │
│                         │
│ 발송 시간: 실시간        │
│ (상황 기반)             │
└─────────────────────────┘
```

### 구성 요소
- 앱 아이콘
- 제목: "장바구니 상품이 곧 품절돼요!" (긴급성)
- 본문: "[상품명] 재고 소진 임박, 지금 구매하세요"
- 이미지: 상품 이미지
- 발송 시간: 실시간 (상황 기반)

## 와이어프레임 4: 알림 설정 페이지

### 목적
사용자가 알림을 직접 관리

### 레이아웃 구조
```
┌─────────────────────────┐
│ 알림 설정               │
├─────────────────────────┤
│                         │
│ 푸시 알림 받기           │
│ [ON/OFF 토글]           │
│                         │
├─────────────────────────┤
│ 알림 카테고리            │
│                         │
│ 할인/특가 알림           │
│ [ON/OFF 토글]           │
│                         │
│ 관심 상품 알림           │
│ [ON/OFF 토글]           │
│                         │
│ 주문/배송 알림           │
│ [ON/OFF 토글]           │
│                         │
│ 이벤트 알림              │
│ [ON/OFF 토글]           │
│                         │
├─────────────────────────┤
│ 알림 받을 시간           │
│                         │
│ 평일                    │
│ 오전 9시 - 오후 10시    │
│ [시간 선택]              │
│                         │
│ 주말                    │
│ 오전 10시 - 오후 11시   │
│ [시간 선택]              │
│                         │
├─────────────────────────┤
│ 알림 빈도                │
│ 하루 최대 알림 개수      │
│ [3개] [5개] [10개]      │
│                         │
│      [저장하기]          │
└─────────────────────────┘
```

### 구성 요소
- 페이지 타이틀: "알림 설정"
- 알림 받기 on/off 토글
- 카테고리별 알림 설정
  - 할인/특가 알림
  - 관심 상품 알림
  - 주문/배송 알림
  - 이벤트 알림
- 알림 받을 시간대 설정
  - 평일 / 주말 구분
  - 시간대 선택 (예: 오전 9시 - 오후 10시)
- 알림 빈도 설정
  - 하루 최대 알림 개수 선택
- [저장하기] 버튼

## 와이어프레임 5: 알림 상세 페이지 (딥링크)

### 목적
알림 클릭 시 정확한 페이지로 이동

### 레이아웃 구조
```
┌─────────────────────────┐
│ [알림에서 이동] 배지     │
├─────────────────────────┤
│                         │
│   [상품 이미지]          │
│                         │
│   상품명                 │
│                         │
│   ~~30,000원~~          │
│   21,000원 (30% 할인)   │
│                         │
│   [구매하기] 버튼        │
│                         │
├─────────────────────────┤
│ 상품 상세 정보           │
│ - 상품 설명              │
│ - 리뷰                  │
│ - Q&A                   │
│                         │
├─────────────────────────┤
│ 이런 상품은 어때요?      │
│ ┌──┐ ┌──┐ ┌──┐         │
│ │사│ │사│ │사│         │
│ │진│ │진│ │진│         │
│ └──┘ └──┘ └──┘         │
│                         │
│ [← 뒤로가기]            │
└─────────────────────────┘
```

### 구성 요소
- 상단: "알림에서 이동" 배지
- 해당 상품/콘텐츠 상세 페이지
- 관련 상품 추천
- [뒤로가기] 버튼

## 와이어프레임 6: 알림 성과 대시보드 (내부용)

### 목적
알림 성과 모니터링

### 레이아웃 구조
```
┌─────────────────────────┐
│ 알림 성과 대시보드       │
├─────────────────────────┤
│                         │
│ 기간 선택: [이번 주 ▼]   │
│                         │
│ ┌──────────────────┐   │
│ │ 발송 건수         │   │
│ │ 5,000,000건      │   │
│ └──────────────────┘   │
│                         │
│ ┌──────────────────┐   │
│ │ 오픈율            │   │
│ │ 18.5%            │   │
│ │ (↑ 10.5%p)       │   │
│ └──────────────────┘   │
│                         │
│ ┌──────────────────┐   │
│ │ 클릭률            │   │
│ │ 5.8%             │   │
│ │ (↑ 3.3%p)        │   │
│ └──────────────────┘   │
│                         │
│ ┌──────────────────┐   │
│ │ 전환율            │   │
│ │ 2.8%             │   │
│ │ (↑ 1.6%p)        │   │
│ └──────────────────┘   │
│                         │
├─────────────────────────┤
│ 세그먼트별 성과          │
│ [그래프]                │
│                         │
│ 시간대별 성과           │
│ [그래프]                │
└─────────────────────────┘
```

### 구성 요소
- 일일/주간/월간 통계
  - 발송 건수
  - 오픈율
  - 클릭률
  - 전환율
- 세그먼트별 성과
- 메시지별 성과 (A/B 테스트 결과)
- 시간대별 성과
- 그래프 및 차트

## 와이어프레임 7: A/B 테스트 결과 페이지 (내부용)

### 목적
메시지 최적화를 위한 테스트 결과 확인

### 레이아웃 구조
```
┌─────────────────────────┐
│ A/B 테스트 결과         │
├─────────────────────────┤
│                         │
│ 테스트 개요              │
│ 기간: 2024.01.01-01.07  │
│ 대상: 신규 사용자       │
│                         │
├─────────────────────────┤
│ A안 (기존)              │
│ 제목: "특가 상품을       │
│       확인하세요!"      │
│                         │
│ 오픈율: 8.0%            │
│ 클릭률: 2.5%            │
│ 전환율: 1.2%            │
│                         │
├─────────────────────────┤
│ B안 (개선)              │
│ 제목: "[사용자명]님,     │
│       관심 상품이        │
│       특가예요!"        │
│                         │
│ 오픈율: 14.8%           │
│ 클릭률: 4.6%            │
│ 전환율: 2.1%            │
│                         │
├─────────────────────────┤
│ 결과                     │
│ ✓ B안 승리              │
│ 오픈율 +85% 향상        │
│ 통계적 유의성: p<0.01   │
│                         │
│ [다음 테스트 계획]       │
└─────────────────────────┘
```

### 구성 요소
- 테스트 개요
  - 테스트 기간
  - 테스트 대상
  - 변형 (A안, B안)
- 성과 지표 비교
  - 오픈율
  - 클릭률
  - 전환율
- 통계적 유의성 표시
- 승리 변형 표시
- [다음 테스트 계획] 버튼
